# Full Stack Learning Roadmap

A beautiful, interactive 4-month roadmap to master **Frontend (React)** and **Express Backend** — from absolute beginner to job-ready full-stack developer.

## Live Demo

Visit the live site: [https://karan00006.github.io/react-roadmap/](https://karan00006.github.io/react-roadmap/)

## What's Inside

- **Phase 1 (Weeks 1–4):** HTML, CSS & JavaScript fundamentals
- **Phase 2 (Weeks 5–12):** React — 2 months of structured learning
- **Phase 3 (Weeks 13–20):** Express & Node.js — 2 months of backend mastery
- Curated free resources (W3Schools, MDN, YouTube channels, official docs)
- Light/Dark mode toggle
- Fully responsive with smooth animations

## Tech Stack

- Pure HTML, CSS & JavaScript — no build step required
- GitHub Pages for hosting
